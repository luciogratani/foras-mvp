# @repo/web — Homepage pubblica

App Next.js 14 (App Router) per la homepage SSR del locale. Gira su porta 3000.

## Dev

```bash
pnpm --filter @repo/web dev   # → http://localhost:3000
```

## Build e type-check

```bash
pnpm --filter @repo/web build
pnpm --filter @repo/web tsc --noEmit
```

Vedi [docs/README.md](../../docs/README.md) per il contesto progetto completo.
